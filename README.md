# python-pyttsx3-tts
Offline Text-to-Speech converter using Python `pyttsx3`.No internet needed.

### Install
```bash
pip install -r requirements.txt
```
### Run
```bash
python pyttsx3Lib.py
```
### Features
- Offline Text-to-Speech converter
- 100% Offline, No internet  needed
- Adjustable speed & volume
